# perica - period tracker

a simple app to track your periods and understand your cycle better. built with privacy in mind - your data stays yours.

## what it does

- track your menstrual cycles and get predictions
- log symptoms, mood, and how you're feeling
- chat with an ai assistant about your health
- see patterns and insights about your body
- everything is private and secure

## how to use it

1. sign up for an account
2. start logging your periods and symptoms
3. the app learns your patterns over time
4. get better predictions and insights

## tech stack

made with react, typescript, and supabase. runs on any device with a web browser.

## getting started

1. clone this repo
2. run `npm install` 
3. add your supabase keys to `.env.local`
4. run `npm run dev`
5. open http://localhost:3000

that's it. no complicated setup, just track your periods and feel more in control of your body.
